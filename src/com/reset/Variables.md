
##VARIABLES

### DECLARATION

`<TYPE> <variableName>;`

### VARIABLE TYPES

#### PRIMITIVE TYPES
* boolean (`true`; `false`)


* numbers
    * integers
      * byte -- 2 bytes
      * short -- 4 bytes
      * int -- 8 bytes
      * long -- 16 bytes
      * char
    * float point
      * float -- with literal f ex.: 15.3f
      * double -- most common
